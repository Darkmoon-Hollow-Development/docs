---
title: "Step 1: Purchase a Minecraft Server"
description: "This is the first step for the BisectHosting server."
---

1. \*\*Navigate \*\*to the [<u>Minecraft Java Server Plans</u>](https://www.bisecthosting.com/selector?plan=minecraft).

<Note>
  BisectOne is an all-in-one server plan, allowing you to switch your server between their large list of available games.
</Note>

2. Select **Buy Plan** or **Help Me Decide** to configure a custom package.
   2. When buying a plan, make sure to check how much RAM you will need to run the server.
2. For **Custom Packages**, **set **the preinstall and player count.


 4. Confirm or change the **Server Location**.
 5. Select a **Billing Cycle**.
    1. Monthly
    2. Quarterly
    3. Semi-Annually
    4. Annually
 6. If present, enter a **Promo Code** and choose **Apply**.
    1. You can use our code **SHADRAGON**
 7. Select any desired **Addons**. 

    <Note>
      The **Free Sticker Pack** requires **BisectBoost**.
    </Note>
 8. Once finished configuring, choose **Buy Plan**.
 9. Review the order and click **Checkout**.
10. Fill in the form and **Payment Details**, then choose **Complete Order**.

      BisectHosting is happy to help with any questions on the server order. Click [<u>here</u>](https://help.bisecthosting.com/)Â to start a conversation with support.
    </Note>
