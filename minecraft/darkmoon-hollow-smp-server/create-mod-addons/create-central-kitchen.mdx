**Automation Chain Example**

```
Wheat → Crusher → Flour  
Flour + Water → Mixer → Dough  
Dough → Oven → Bread  
Bread → Depot → Storage
```

**Tips**

- Use Smart Chutes for portion control
- Excellent for colony kitchens and markets
